SELECT* Studio

Feed it your database. Click your sentence together. Read the SQL.

01

Teach it your database

Paste your SQL dump or upload the .sql file — SELECT, UPDATE, DELETE and ALTER all work on it. Only the CREATE TABLE structure is read, so a dump with or without data works fine. Foreign keys are detected automatically.

02

Build your sentence

Click pieces together like Lego. The connector words are the SQL clauses in disguise: from = FROM, where = WHERE, for each = GROUP BY, sorted by = ORDER BY. Hover any piece to see its SQL light up.

⇧ Load a database in step 1 first (or hit the demo button).
Things to try (with the demo library)
  1. The 3 most expensive books.
  2. Each book together with its author's name (a join).
  3. The number of books per genre.
  4. Books that cost more than the average book price.
  5. Members who still have a book out (loan not returned yet).
